# Internationalization (i18n)

> **Looking for the full contributor guide?**  
> See **[docs/localization-guide.md](./localization-guide.md)** for the complete workflow: adding locales, translating strings, RTL support, pluralization, and the new-locale checklist.

Fund-My-Cause uses [next-intl](https://next-intl.dev) for internationalization. Translation files live in `apps/interface/messages/`.

## Supported Languages

| Code | Language | RTL |
|------|----------|-----|
| `en` | English  | No  |
| `es` | Español  | No  |
| `fr` | Français | No  |
| `de` | Deutsch  | No  |
| `zh` | 中文     | No  |
| `ar` | العربية  | Yes |
| `he` | עברית    | Yes |

## Quick Start

### Add a new locale

1. Add the locale code to `src/i18n/config.ts`:
   ```ts
   export const locales = ["en", "es", "fr", "de", "zh", "ar", "he", "pt"] as const;
   export const localeNames: Record<Locale, string> = {
     // ...existing entries...
     pt: "Português",
   };
   ```
   If RTL, add to `rtlLocales` as well.

2. Create `messages/pt.json` by copying `messages/en.json` and translating every value.

3. That's it — the middleware, layout, and language selector pick up the new locale automatically.

For detailed instructions including RTL considerations, pluralization, and testing, see the [Localization Contributor Guide](./localization-guide.md).

## File Structure

```
apps/interface/
├── messages/
│   ├── en.json   ← source of truth
│   ├── es.json
│   ├── fr.json
│   ├── de.json
│   ├── zh.json
│   ├── ar.json
│   └── he.json
└── src/i18n/
    ├── config.ts      ← locale list, RTL locales, display names
    ├── routing.ts     ← next-intl routing (localePrefix: "as-needed")
    ├── request.ts     ← server-side message loading
    └── navigation.ts  ← typed Link/useRouter/usePathname
```

## Using Translations

**Server components:**
```tsx
import { useTranslations } from "next-intl";
const t = useTranslations("home");
return <h1>{t("heroTitle")}</h1>;
```

**Client components:**
```tsx
"use client";
import { useTranslations } from "next-intl";
const t = useTranslations("campaigns");
return <button>{t("filterAll")}</button>;
```
